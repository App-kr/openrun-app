## 스크린샷 촬영 가이드 (5장 필수)

> 기기: Pixel 8 또는 Galaxy S23 (1080×2400 권장)
> Play Store 요구 사양: 최소 320px, 최대 3840px (세로/가로 비율 16:9 또는 9:16)

| # | 화면 | 촬영 방법 |
|---|------|-----------|
| 1 | **공연 목록** | 앱 실행 → 필터 '클래식' 선택 → 카드 3~4개 보이는 상태에서 캡처 |
| 2 | **공연 상세** | 공연 카드 탭 → 포스터가 있는 공연 선택 → 예매 버튼까지 보이도록 캡처 |
| 3 | **알림 설정** | 공연 카드 하단 🔔 버튼 탭 → 바텀시트 '1시간 전' 선택 상태 캡처 |
| 4 | **알림 목록** | /alarms 화면 → 알람 설정된 공연 목록 캡처 |
| 5 | **온보딩** | 설정 > 온보딩 초기화 → 인트로 2페이지('알림') 캡처 |

## Play Console 제출 순서

1. [ ] [Play Console](https://play.google.com/console) 로그인
2. [ ] 앱 만들기 → 패키지명: `com.taekit.app`
3. [ ] 스토어 등록 정보 → 위 텍스트 붙여넣기
4. [ ] 스크린샷 5장 업로드 (1080×2400 PNG)
5. [ ] 고해상도 아이콘 업로드 (512×512 PNG, 투명 배경)
6. [ ] 개인정보처리방침 URL 입력: `https://openrun-api.onrender.com/privacy`
7. [ ] 데이터 보안 섹션 작성 (위 표 참고)
8. [ ] 콘텐츠 등급 설문 완료
9. [ ] 프로덕션 트랙 → AAB 업로드: `build/app/outputs/bundle/release/app-release.aab`
10. [ ] 검토 제출 → 보통 1~3일 소요
